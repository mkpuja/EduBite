# EduBite

EduBite is a Django-based food ordering web application for students and teachers.

## Features

* User Registration & Login
* Dynamic Food Menu
* Food Search
* Category Filtering
* Add to Cart
* Quantity Management
* Place Order
* Cancel Order
* Order Status Tracking
* Admin Order Management

## Technology Used

* Python
* Django
* SQLite
* HTML
* CSS
* Bootstrap
